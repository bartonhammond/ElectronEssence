### Setup

```

1) git clone git@github.com:bartonhammond/ElectronEssence.git
2) cd ElectronEssence
3) npm install
4) npm run build-dev
In another terminal:
5) npm start

```
Make edits and use Reload (cmd-r) to reload changes
